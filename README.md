#  Tích hợp vbot-todo sử dụng uni-app vào Android Native

Repo này trình bày phương pháp tích hợp vbot-todo sử dụng uni-app vào một ứng dụng Android Native.

---
##  Cài đặt
Dưới đây là hướng dẫn để clone và chạy cả phiên bản demo.
### Tải mã nguồn (Clone)
Tải repo:
```text
git clone https://github.com/HoangHuuTinIT/vbot-todo-uni-app.git
```
## Chạy Demo
### Cách chạy Demo
1. Mở Android Studio.

2. Chọn File > Open > Chọn thư mục được clone về > chọn OK

4. Đợi Android Studio đồng bộ (Sync) Gradle.

5. Nhấn nút "Run 'app'" để build và chạy ứng dụng.
